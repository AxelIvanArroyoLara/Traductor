# User flows

En el siguiente diagrama de flujo de usuario se detalla la arquitectura y el recorrido del usuario dentro de la web.

* Toma de Decisiones: Los diamantes en el diagrama representan puntos donde el sistema evalúa condiciones, dependiendo de la respuesta, el usuario es redirigido a una acción distinta.

* Módulos en Paralelo: Existen bloques de procesos agrupados que indican que el usuario tiene múltiples opciones disponibles simultáneamente o que el sistema procesa varios datos al mismo tiempo.

* Manejo de Errores: El sistema permite regresar a pasos anteriores para corregir los datos sin perder el progreso total.

* Finalización: Los puntos marcados señalizan los estados finales, ya sea una tarea completada con éxito o una salida del sistema por error o cancelación.
